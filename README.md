## 🚀 Potential Further Steps

To further enhance both UX/UI and implementation beyond the scope of a limited-time project, here are some potential future measures I would consider in a production landscape.

* Adapt and implement mega menu for mobile screens
* Small icon animations in the button to make them feel more dynamic and highlight them more -> can boost UI and UX
* For bigger project scope: refactor Slider component into a custom hook to increase scalability
* Further unit / integration tests for high-logic components
